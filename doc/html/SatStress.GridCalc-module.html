Module GridCalc

source code

Calculate stresses on an icy satellite over a rectangular geographic region on a regularly spaced lat-lon grid.

The datacube containing the results of the calculation are output as a Unidata netCDF (.nc) file, which can be displayed using a wide variety of visualization software.
